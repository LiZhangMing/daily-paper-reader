## 摘要
投机解码（Speculative decoding）加速了大语言模型（LLM）的推理，但当目标模型针对特定领域进行微调时，其性能会下降。一种朴素的解决方案是为每个目标模型重新训练草稿模型，但这既昂贵又低效。为解决这一问题，我们引入了一个名为“高效草稿适配”（Efficient Draft Adaptation，简称 EDA）的参数和数据高效框架，用于高效地适配草稿模型。EDA 引入了三项创新：(1) 一种解耦架构，利用共享和私有组件分别对共享和目标特定的输出分布进行建模，通过仅更新轻量级的私有组件来实现参数高效的适配；(2) 一种数据再生策略，利用微调后的目标模型重新生成训练数据，从而改善训练与投机解码之间的对齐，提高平均接受长度；(3) 一种样本选择机制，优先选择高价值数据以实现高效适配。我们的实验表明，EDA 有效地恢复了微调模型上的投机性能，与完全重新训练相比，在显著降低训练成本的同时，实现了更优的平均接受长度。代码可在 https://github.com/Lyn-Lucy/Efficient-Draft-Adaptation 获取。

## Abstract
Speculative decoding accelerates LLM inference but suffers from performance degradation when target models are fine-tuned for specific domains. A naive solution is to retrain draft models for every target model, which is costly and inefficient. To address this, we introduce a parameter- and data-efficient framework named Efficient Draft Adaptation, abbreviated as EDA, for efficiently adapting draft models. EDA introduces three innovations: (1) a decoupled architecture that utilizes shared and private components to model the shared and target-specific output distributions separately, enabling parameter-efficient adaptation by updating only the lightweight private component;(2) a data regeneration strategy that utilizes the fine-tuned target model to regenerate training data, thereby improving the alignment between training and speculative decoding, leading to higher average acceptance length;(3) a sample selection mechanism that prioritizes high-value data for efficient adaptation. Our experiments show that EDA effectively restores speculative performance on fine-tuned models, achieving superior average acceptance lengths with significantly reduced training costs compared to full retraining. Code is available at https://github.com/Lyn-Lucy/Efficient-Draft-Adaptation.

*   **研究动机**：投机解码（Speculative Decoding）是加速大语言模型（LLM）推理的主流技术，其效率高度依赖于“草稿模型”与“目标模型”输出分布的对齐程度。
*   **核心矛盾**：在实际应用中，目标模型常针对特定领域（如数学、代码、医疗）进行微调（SFT），这会导致其分布发生偏移。此时，原有的通用草稿模型会因“失配”导致接受率骤降。
*   **挑战**：为每个微调后的目标模型重新训练一个完整的草稿模型成本极高且效率低下。因此，如何以极低的计算和数据成本，让草稿模型快速适配新的目标模型，是本文的核心课题。

### 2. 方法论（EDA 框架）
EDA 框架从架构、数据生成和样本筛选三个维度提出了创新：
*   **解耦架构（Shared-Private Gated Architecture）**：
    *   将草稿模型中的 FFN 层替换为“共享专家”和“私有专家”的门控组合。
    *   **共享专家**：继承自预训练草稿模型并保持冻结，负责捕捉通用的语言规律。
    *   **私有专家**：轻量级模块，在适配过程中进行训练，负责捕捉目标模型微调后的特定领域偏好。
    *   通过一个可学习的门控机制动态调节两者的贡献。
*   **领域特定自生成策略（Domain-Specific Self-Generation）**：
    *   为了消除训练数据与实际推理分布之间的不一致，EDA 使用微调后的目标模型在领域数据集上进行自回归生成，产生“自生成数据集”。
    *   草稿模型直接学习目标模型生成的 Token 序列，从而在推理阶段能更精准地预测目标模型的行为。
*   **基于表示偏移的样本选择（Data-Efficient Selection）**：
    *   利用 PCA（主成分分析）降低目标模型隐藏层表示的维度。
    *   计算样本在表示空间中相对于通用分布的马氏距离（Mahalanobis distance）。
    *   优先选择那些“偏离通用分布最远”的样本进行训练，因为这些样本包含了最多的领域特定信息。

### 6. 主要结论与发现
*   **性能恢复**：EDA 显著提升了平均接受长度（$\tau$）。例如在数学任务上，EDA 将 $\tau$ 从 1.19（未适配）提升至 5.19，甚至超过了全参数微调（4.69）。
*   **加速效果**：在各领域均实现了 2.5x 至 3.5x 的端到端解码加速。
*   **数据高效性**：使用 PCA 筛选出的 **50% 数据** 即可达到使用 100% 数据约 99% 的性能，证明了并非所有微调数据对草稿模型适配都有同等价值。
*   **架构优势**：共享-私有解耦架构比单纯的 LoRA 或全量微调更能有效保留通用能力并吸收领域知识。

### 7. 优点（亮点）
*   **双重高效**：同时实现了参数高效（只更新私有专家）和数据高效（样本筛选）。
*   **即插即用**：不需要访问目标模型的原始微调数据（通过自生成解决），这在处理私有或闭源模型时非常实用。
*   **训练-推理一致性**：通过自生成数据解决了投机解码中长期存在的“训练目标与推理任务不匹配”的问题。

### 8. 不足与局限
*   **模型覆盖面**：实验主要集中在 Qwen2.5-7B 系列模型上，虽然代表性强，但未在更大规模（如 70B+）或其他架构（如 Llama 3, Mistral）上验证泛化性。

*   **专家数量固定**：目前仅使用了双专家架构（一共享一私有），在面对多任务或更复杂的混合领域时，可能需要更复杂的 MoE 架构。
