## 摘要
测试时自适应 (TTA) 已被广泛研究，旨在防止测试数据与训练分布不一致时出现的性能下降。然而，如何以最少的参数更新充分利用大型预训练模型的丰富表示仍有待探索。在本文中，我们提出了内置谱专家混合 (IMSE)，它利用了 Vision Transformer 中固有嵌入的谱专家。我们通过奇异值分解 (SVD) 对每个线性层进行分解，仅调整奇异值，同时保持奇异向量固定。我们进一步确定了 TTA 中熵最小化的一个关键局限性：它通常会导致特征崩溃，使模型依赖于领域特定特征而非类别判别特征。为了解决这个问题，我们提出了一种基于专家-输入对齐的多样性最大化损失，鼓励在自适应过程中多样化地利用谱专家。在持续测试时自适应 (CTTA) 场景中，除了保留预训练知识外，保留并重用来自先前观察到的领域的知识也至关重要。我们引入了领域感知谱代码检索，通过估计输入分布来检测领域偏移，并检索自适应后的奇异值以实现快速自适应。因此，我们的方法在 TTA 设置下的各种分布偏移基准测试中实现了最先进的性能。在 CTTA 和渐进式 CTTA 中，它分别将准确率提高了 3.4 个百分点 (pp) 和 2.4 个百分点，同时所需的可训练参数减少了 385 倍。我们的代码可在 https://github.com/baek85/IMSE 获取。

## Abstract
Test-time adaptation (TTA) has been widely explored to prevent performance degradation when test data differ from the training distribution. However, fully leveraging the rich representations of large pretrained models with minimal parameter updates remains underexplored. In this paper, we propose Intrinsic Mixture of Spectral Experts (IMSE) that leverages the spectral experts inherently embedded in Vision Transformers. We decompose each linear layer via singular value decomposition (SVD) and adapt only the singular values, while keeping the singular vectors fixed. We further identify a key limitation of entropy minimization in TTA: it often induces feature collapse, causing the model to rely on domain-specific features rather than class-discriminative features. To address this, we propose a diversity maximization loss based on expert-input alignment, which encourages diverse utilization of spectral experts during adaptation. In the continual test-time adaptation (CTTA) scenario, beyond preserving pretrained knowledge, it is crucial to retain and reuse knowledge from previously observed domains. We introduce Domain-Aware Spectral Code Retrieval, which estimates input distributions to detect domain shifts, and retrieves adapted singular values for rapid adaptation. Consequently, our method achieves state-of-the-art performance on various distribution-shift benchmarks under the TTA setting. In CTTA and Gradual CTTA, it further improves accuracy by 3.4 percentage points (pp) and 2.4 pp, respectively, while requiring 385 times fewer trainable parameters. Our code is available at https://github.com/baek85/IMSE.

论文关注的是**测试时自适应（Test-Time Adaptation, TTA）**，即当部署的模型遇到与训练集分布不同的测试数据时，如何在不访问原始训练数据的情况下在线调整模型。
*   **研究动机**：
    *   **参数效率低下**：现有方法要么只更新极少数参数（如 Norm 层），限制了表达能力；要么引入额外的模块（如 Adapters），增加了推理开销。
    *   **特征坍缩（Feature Collapse）**：常用的“熵最小化”损失在无监督自适应时，容易使模型过度关注领域特定特征（Domain-specific）而忽略类别判别特征（Class-discriminative），导致特征多样性丧失。
    *   **持续学习中的遗忘**：在持续 TTA（CTTA）场景下，模型容易遗忘之前学到的领域知识，且难以快速适应新领域。

### 2. 方法论：核心思想与关键技术
论文提出了 **IMSE（内置谱专家混合）** 框架，核心思想是将预训练模型的线性层重新解释为多个“谱专家”的混合。

*   **内置谱专家（Intrinsic Spectral Experts）**：
    *   利用**奇异值分解（SVD）**将线性层权重 $W$ 分解为 $U \Sigma V^\top$。
    *   将每个秩为 1 的组件 $u_i v_i^\top$ 视为一个“谱专家”，而对应的奇异值 $\sigma_i$ 则是该专家的贡献权重。
    *   **技术细节**：在自适应过程中，固定正交基（奇异向量 $U, V$），仅微调奇异值 $\Sigma$（称为 Spectral Code）。这保留了预训练的特征提取空间，同时实现了极高的参数效率。
*   **多样性最大化损失（Diversity Maximization Loss, $L_{dm}$）**：
    *   通过计算专家与输入之间的对齐统计量（均值和标准差），引入 $L_{dm}$ 来鼓励不同谱专家的多样化激活，从而补偿熵最小化导致的特征坍缩。
*   **领域感知谱代码检索（Domain-Aware Spectral Code Retrieval）**：
    *   **领域描述符**：提取 Patch Token 的通道均值和方差作为分布估计。
    *   **检索机制**：使用对称 KL 散度检测领域偏移。当检测到新领域时，从“领域库（Domain Bank）”中检索最相似的已存奇异值进行初始化，实现快速适应并减少遗忘。

### 6. 主要结论与发现
*   **性能领先**：在 CTTA 场景下，IMSE 比之前的 SOTA 方法（ViDA）准确率提升了 **6.7pp**；在渐进式偏移下提升了 **2.4pp**。
*   **谱专家的重要性**：实验发现，**大奇异值**对应的谱专家在自适应中起主导作用，微调 Top-R% 的奇异值效果远好于微调 Bottom-R%。
*   **多样性的价值**：证明了维持特征多样性是提升无监督自适应性能的关键，$L_{dm}$ 能有效防止模型陷入领域特定的局部最优。

### 7. 优点（亮点）
*   **极高的参数效率**：通过 SVD 巧妙地将复杂的权重更新简化为一维向量（奇异值）的调整。
*   **即插即用与通用性**：适用于各种预训练模型（MAE, CLIP 等）和不同规模的 ViT 架构。
*   **知识重用机制**：Domain Bank 机制解决了 CTTA 中“既要适应新领域，又要保留旧知识”的矛盾，且检索开销极低。

### 8. 不足与局限
*   **存储开销**：虽然可训练参数极少，但为了推理时重构权重，需要存储分解后的正交基矩阵 $U$ 和 $V$，这会增加模型在内存/显存中的原始占用（相比于未分解的 $W$）。
*   **领域库增长**：在极端多变的场景下，Domain Bank 的条目可能会持续增加，虽然单个条目很小，但长期维护仍需考虑清理策略。
*   **架构限制**：目前主要针对 Transformer 架构中的线性层进行优化，对于 CNN 等其他架构的适用性（如卷积层的 SVD 分解）讨论较少。
